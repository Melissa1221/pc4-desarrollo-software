# PC4 - Desarrollo de Software

Plataforma de rescate y adopción de mascotas. Núcleo de dominio en Java
que implementa los tres módulos pedidos usando patrones de diseño.

## Módulos

1. Reporte de animales perdidos y alertas
2. Buscador por imagen con tres intenciones (adopción, venta, verificar pérdida)
3. Red de cuidadores

## Ejecutar

```
mvn compile
mvn exec:java
```

Ver `INFORME.md` para requisitos cumplidos, patrones y arquitectura.
